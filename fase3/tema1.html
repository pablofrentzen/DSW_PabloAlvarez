Frameworks CSS

Losframework son un conjunto de herramientas, librerías, convenciones y buenas
prácticas que pretenden encapsular las tareas repetitivas en módulos genéricos
fácilmente reutilizables.
De la misma forma, un framework CSS es un conjunto de herramientas, hojas de estilos
y buenas prácticas que permiten al diseñador web olvidarse de las tareas repetitivas
para centrarse en los elementos únicos de cada diseño en los que puede aportar valor.
Los frameworks CSS más completos incluyen utilidades para que el diseñador no tenga
que trabajar en ningún aspecto genérico del diseño web.
